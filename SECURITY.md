# Security Policy

To maintain the security of this repository, we have established the following policy.


## Supported Versions

Only the latest version of this project is supported.


## Reporting a Vulnerability

If you identify a security vulnerability, please report it privately using the following procedure:

1. For security reasons, refrain from creating a public issue.


2. Navigate to the **"Security and Quality"** tab of the repository page.


3. Click the **"Report a vulnerability"** button to submit your report privately.

Reports can be submitted in either Japanese or English.
We will acknowledge your report within 48 hours.

---

# セキュリティポリシー

このリポジトリの安全性を維持するため、以下のポリシーを公開しています。


## サポートバージョン

最新バージョンのみサポート対象となります。


## 脆弱性の報告について

セキュリティ上の脆弱性を発見された場合は、以下の手順で非公開でご報告いただけますようお願いいたします。

1. セキュリティ上の理由から、公開Issueを作成しないでください。
2. リポジトリページの **"Security and Quality"** タブに移動してください。
3. **"Report a vulnerability"** ボタンをクリックして、非公開で報告を送信してください。

ご報告は日本語・英語どちらでも構いません。ご報告後、48時間以内に確認のご連絡をさせていただきます。
